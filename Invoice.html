INVOICE

The Invoice submenu within the Account menu allows users to create and manage invoices for products or services provided to customers. Users can generate professional-looking invoices, specify item details, set payment terms, and track payment status. This feature streamlines the invoicing process and helps ensure timely and accurate billing.

Add Invoice

To add Invoice:
1. Click the + sign to launch the Add Invoice window
2. Select the preferred shipper, cosignee and supplier as the case maybe
3. Select and fill all other relevant field
4. Click Save to save receipt

Note after save the number of receipt saved icreases and the $ sign means invoice yet to be created
